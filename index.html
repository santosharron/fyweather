Get Started

The FYWeather API provides programmatic access to the weather data provided by OpenWeather API.

To use this API, you don't need any API key.

API Endpoint
https://santosharron.github.io/fyweather/

To get Data from API you need to make a POST call to the following urls,

There Are 4 API calls available:

  1. Get City Data
  2. Get nearby City from Coordinates
  3. Get Weather from city ID
  4. Get Weather from Coordinates


Get Data

Get City Data


To get city data you need to make a POST call to the following url :


https://santosharron.github.io/fyweather/city/query



JSON PARAMETRS


Field Type Description
city string matching city name


Get nearby City from Coordinates


To get a city nearby a specific coordinates you need to make a POST call to the following url :


https://santosharron.github.io/fyweather/city/coord



JSON PARAMETRS


Field Type Description
lat number Location Latitude Coordinates
lon number Location Longitude Coordinates


Get Weather by City Id


To get the current weather in a city you need to make a POST call to the following url :


https://santosharron.github.io/fyweather/weather/id



JSON PARAMETRS


Field Type Description
id string location Id


Get Weather from Coordinates


To get the full weather data in a city near you, you need to make a POST call to the following url :


https://santosharron.github.io/fyweather/weather/coord/full



JSON PARAMETRS


Field Type Description
lat number Location Latitude Coordinates
lon number Location Longitude Coordinates


# Here is a curl example

curl --header "Content-Type: application/json" \
--request POST \
--data '{"city":"newyork"}' \
https://santosharron.github.io/fyweather/city/query
